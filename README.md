# Advanced (Digital) Typography
This is the repository of the educational typography project *advanced (digital) typography* which took place at the FH Potsdam in 2025 under the guidance of Felix Walser.

## Information
During a one week workshop, we explored new forms of typographic systems with InDesign scripts. We published these scripts here for further use and edit.
